Variáveis Compostas

O que são?

Variáveis compostas devem ser capazes de armazenar vários valores em uma mesma estrutura.

Composições

Variáveis compostas também são chamadas de 'array'(vetor).
As arrays são organizadas com colchetes.
Toda array possui índices, que são as posições.
Toda array é composta por elementos.
E dentro dos elementos se tem um conteúdo/valor.

Como criar em JS

Para criar, utilizamos o comando 'let'.
Exemplo:
let x = []

Alguns comandos

---Criando um vetor---

let num = [5, 8, 2, 9, 3]

---Pegar um valor--

num[posição]

---Adicionando um novo elemento ao vetor em uma posição específica---

num[5] = 6
// o vetor agora é [5, 8, 2, 9, 3, 6]

---Adicionando um elemento na última posição---

num.push(7)
// o vetor agora é [5, 8, 2, 9, 3, 6, 7]

---Como saber o comprimento de um vetor---

num.length
// o tamanho do vetor é 7

---Colocando os elementos em ordem crescente---

num.sort()
// o vetor agora é [2, 3, 5, 6, 7, 8, 9]

---Mostrando o vetor sem a formatação padrão---

for(let i = 0; i < num.length; i++){
console.log(num[i])
}
// com isso, aparecerá os valores do vetor um abaixo do outro

---Forma simplificada de percorrer o vetor---

for(let i in num){
console.log(num[i])
}

---Buscar valor dentro de um vetor---

num.indexOf(valor_desejado)
// com isso, aparecerá a posição em que esse valor se encontra
