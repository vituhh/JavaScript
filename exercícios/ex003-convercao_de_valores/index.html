Como converter valores

Para converter de String para Number, existem três formas:

  • Number.parseInt(n)
  • Number.parseFloat(n)
  • Number(n)

Para converter de Number para String, existem daus formas:

  • String(n)
  • n.toString()

Template string

Ao invés de digitar
window.alert("A soma dos valores é " + soma)
digite
window.alert("A soma dos valores é ${soma}")
